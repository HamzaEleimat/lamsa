## Production Guidelines

### Animation Standards:

#### Transitions:
- **Fade In/Out**: 0.5 seconds for scene transitions
- **Slide Animations**: Left-to-right for progression, right-to-left for Arabic
- **Scale Effects**: 1.05x zoom for emphasized elements
- **Color Highlights**: Yellow (#FFD700) for important buttons/fields

#### Text Effects:
- **Typing Animation**: 50ms per character for form filling
- **Counter Animation**: Smooth number increments for prices/durations
- **Progress Bars**: Linear fill animation at 2 seconds duration
- **Checkmark Animation**: Bounce effect on completion

### Audio Production:

#### Music Selection:
- **Intro/Outro**: Upbeat, professional background music
- **Tutorial Sections**: Subtle, non-distracting ambient sounds
- **Success Moments**: Short celebratory chimes
- **Transitions**: Soft swoosh sounds between sections

#### Voiceover Guidelines:
- **Pace**: 150-160 words per minute for clear comprehension
- **Tone**: Professional, encouraging, confident
- **Emphasis**: Highlight key terms and action items
- **Pauses**: Allow time for visual processing between concepts

### Visual Consistency:

#### Color Scheme:
- **Primary**: BeautyCort Purple (#6B46C1)
- **Secondary**: Orange (#ED8936) for accents
- **Success**: Green (#48BB78) for completed actions
- **Warning**: Yellow (#F6AD55) for important notices
- **Text**: Dark Gray (#2D3748) for readability

#### Typography:
- **Headings**: Inter Bold, 24pt
- **Body Text**: Inter Regular, 16pt
- **Arabic Text**: Tajawal, appropriate weights
- **Captions**: Inter Medium, 14pt
- **Fine Print**: Inter Regular, 12pt

### Screen Recording Standards:

#### Device Setup:
- **Resolution**: 1080p minimum for crisp text
- **Orientation**: Portrait for mobile-first experience
- **Frame Rate**: 30fps for smooth motion
- **Device**: iPhone or high-end Android for consistency

#### Interface Preparation:
- **Clean State**: Remove notifications, test data
- **Consistent Timing**: Practice gestures for smooth demonstration
- **Realistic Data**: Use Jordan-appropriate names and locations
- **Error-Free**: No typos or incorrect information

This comprehensive storyboard framework ensures consistent, professional video production that effectively guides providers through BeautyCort platform usage while maintaining engagement and clarity.
